# 🧠 Calculator and Train Seat Layout Project
This project is a collection of two simple web applications: a calculator and a train seat layout viewer. The calculator application allows users to perform basic arithmetic operations, while the train seat layout viewer displays different coach layouts and allows users to select seats. The project is built using HTML, CSS, and JavaScript, and utilizes Bootstrap for styling and layout.

## 🚀 Features
- Basic arithmetic operations (addition, subtraction, multiplication, division) in the calculator application
- Selection of different coach layouts in the train seat layout viewer
- Display of seat arrangements for each coach
- User-friendly interface for both applications
- Custom styling using CSS

## 🛠️ Tech Stack
* Frontend: HTML, CSS, JavaScript
* Styling: Bootstrap
* Libraries: Bootstrap CSS and JavaScript
* Local files: style.css, script.js

## 📦 Installation
To get started with the project, follow these steps:
1. Clone the repository to your local machine
2. Open the `calculator` or `Train boxes` folder in your preferred code editor
3. Ensure you have a code editor or IDE with HTML, CSS, and JavaScript support
4. Open the `index.html` file in a web browser to view the application

## 💻 Usage
1. Open the `calculator/index.html` file in a web browser to use the calculator application
2. Select two numbers and an operation using the input fields and dropdown menu
3. Click the "Calculate" button to perform the operation and display the result
4. Open the `Train boxes/index.html` file in a web browser to use the train seat layout viewer
5. Click on a coach button to display the corresponding seat layout
6. View the seat arrangement and select seats as needed
